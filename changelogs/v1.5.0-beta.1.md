## v1.5.0-beta.1 — 2026-07-06

First beta of the v1.5.0 series since [v1.4.1](https://github.com/chenhg5/cc-connect/releases/tag/v1.4.1)
stable. **Three new integration surfaces** join the family: Tencent Yuanbao Bot API,
self-hosted `cloud_web` IM gateway, and the Reasonix agent adapter — plus Pi RPC mode,
Russian i18n, per-session agent idle timeout, and a batch of platform/core fixes.

---

## New Platforms

- **Tencent Yuanbao Bot API** — first-class Yuanbao platform adapter with bot identity,
  message routing, and reply support (#1445,
  [@skyblue](https://github.com/skyblue)).
- **cloud_web** — self-hosted IM gateway platform using CWIP v1 protocol (websocket /
  long_poll / gateway transports, full inbound/outbound + capability negotiation +
  graceful degradation). See `docs/cloud-web.md` (#1282,
  [@jiagou123](https://github.com/jiagou123)).

cc-connect now ships **17+ chat platforms** (exact count depends on build tags).

---

## New Agents

- **Reasonix** — HTTP serve API adapter bridging Reasonix multi-model coding agent
  (POST /submit, SSE /events, POST /approve). Supports default/yolo/plan permission
  modes, SSE auto-reconnect with backoff, and thinking accumulator (#1281,
  [@mchenziyi](https://github.com/mchenziyi)).

---

## Features

### Agent lifecycle
- **`agent_session_idle_timeout_mins`** — per-project config to close idle live agent
  processes after a clean turn while preserving the cc-connect session and saved agent
  session ID. The next message starts a new agent process and resumes the same
  conversation. Set to `0` or leave unset to disable (#1338,
  [@hl1221hl](https://github.com/hl1221hl)).

### Pi
- **RPC mode** — RPC mode with `extension_ui` permission forwarding (#1440,
  [@happyTonakai](https://github.com/happyTonakai)).

### Feishu
- **`mention_map`** — outbound bot-to-bot `@` resolution via configurable name →
  open_id mapping. When an agent writes `@BOT-A` in its reply, cc-connect converts it
  to a native Feishu `<at>` tag (#1341 fixing #1322,
  [@generspooler](https://github.com/generspooler)).
- **Inter-bot relay** — route inter-bot visibility echoes back into caller's Feishu
  thread (#1413, [@zhangshuaimk-boop](https://github.com/zhangshuaimk-boop)).

### i18n
- **Russian (ru) locale** — Web admin UI Russian translations (#1449,
  [@sonsay](https://github.com/sonsay)).

## Bug Fixes

- **display**: hide agent footer lines in progress output (#1416,
  [@AaronZ345](https://github.com/AaronZ345)).
- **core**: resume stream preview after permission prompt resolves (#1451,
  [@happyTonakai](https://github.com/happyTonakai)).
- **core**: always emit absolute paths from `SaveFilesToDisk` / `AppendFileRefs` —
  fixes relative `work_dir` attachments silently dropped by spawned agent processes
  (#1462 fixing #1459, [@chenhg5](https://github.com/chenhg5)).
- **core**: create queue placeholder before session lock — prevents concurrent message
  queue miss under load (#1389, [@xxb](https://github.com/xxb)).
- **codex**: time out blocked app-server writes (#1448,
  [@AaronZ345](https://github.com/AaronZ345)).
- **slack**: suppress `NO_REPLY` marker on streaming-card silent replies (#1397,
  [@spinsirr](https://github.com/spinsirr)).
